# Update Skill Graph

## 실행 순서

### 1. 즉시 확인 응답
텔레그램으로 "스킬 그래프 업데이트를 시작합니다 ⏳" 전송.

### 2. 기존 스킬 그래프 읽기
- `data/profile/skill_graph.md` 읽기 (이전 기준선)
- `data/profile/runner_profile.md` 읽기 (현재 목표)

### 3. 최근 데이터 수집

최근 4주 데이터를 중심으로, 전체 트렌드와 비교:

```bash
# 최근 4주 활동 조회
python scripts/garmin_sync.py --range <4주전> <오늘>
```

추가로 수집:
- `data/training/` 최근 4~8주 주간 리포트
- `data/health/` 최근 2~3개월 건강 데이터
- `data/races/` 최근 대회 기록 (있는 경우)

### 4. 분석 & 업데이트

`data/profile/skill_graph.example.md` 템플릿의 각 섹션을 업데이트:

#### 볼륨 프로필
- 최근 4주/12주 평균 주간 거리, 세션 수 재계산
- 볼륨 트렌드 방향 업데이트
- 10% 룰 준수율 재계산

#### 페이스 프로필
- 존별 페이스 범위 업데이트
- 80/20 비율 재계산
- 이지런 페이스 트렌드 업데이트

#### 심박 프로필
- RHR/HRV 평균 및 트렌드 업데이트
- 심박 존 시간 분포 재계산

#### 건강 & 회복
- 평균 수면, 체중, 스트레스 업데이트

#### 레이스 예측
- Garmin 최신 예측 기록 반영
- 사용자 목표 대비 갭 재계산

#### 강점 & 약점
- 이전 약점이 개선되었는지 평가
- 새로운 강점/약점 식별

#### 코칭 권장사항
- 단기/중기/장기 방향 업데이트
- 대회 일정 기반 우선순위 조정

### 5. 변경 이력 기록
`skill_graph.md` 하단 변경 이력 테이블에 오늘 날짜와 업데이트 요약 추가.

### 6. 이전 vs 현재 비교 요약

텔레그램으로 주요 변화를 전달:

```
📈 스킬 그래프 업데이트 완료

🔄 주요 변화 (이전 → 현재)
- 주간 평균 거리: Xkm → Xkm
- 이지런 페이스: X'XX" → X'XX"/km
- HRV 평균: Xms → Xms
- RHR: Xbpm → Xbpm

💪 개선된 영역
- [구체적 개선 내용]

🎯 다음 목표
- [권장사항 요약]
```
