# Skill Graph

> 이 파일은 초기 설치 시 1년치 Garmin 데이터를 분석한 뒤 LLM이 자동으로 작성합니다.
> 이후 주기적으로 업데이트되며, 훈련 계획 수립의 기반 데이터로 사용됩니다.

## 분석 기간

- **시작**: YYYY-MM-DD
- **종료**: YYYY-MM-DD
- **총 활동 수**: N회
- **총 거리**: N km
- **분석 생성일**: YYYY-MM-DD
